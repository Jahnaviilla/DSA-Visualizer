# 🚀 DSA Visualizer Portal

An interactive **Data Structures and Algorithms Visualizer** built using **HTML, CSS, and JavaScript**. This project provides an intuitive and visually appealing platform to understand how different data structures and algorithms work through real-time animations and interactive operations.

---

## 📌 Project Overview

The DSA Visualizer Portal helps students and beginners learn Data Structures and Algorithms by providing graphical visualizations of various operations. Users can perform operations such as insertion, deletion, searching, sorting, and traversal while observing the internal working step-by-step.

The application features a modern dashboard interface, execution logs, and theory sections to improve both practical understanding and conceptual knowledge.

## 🚀 How to Run the Project

### Method 1

1. Clone the repository

```bash
git clone https://github.com/YourUsername/DSA-Visualizer.git
```

2. Open the project folder.

3. Double-click `index.html`

OR

4. Open the folder in Visual Studio Code.

5. Install the **Live Server** extension.

6. Right-click on `index.html`

7. Click **Open with Live Server**
